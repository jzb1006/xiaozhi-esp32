## 麦克风接线

INMP441 使用 3.3 V 供电，接到 ESP32-S3 的 I2S 输入引脚。

| INMP441 | ESP32-S3 | 说明 |
| --- | --- | --- |
| VDD | 3V3 | 麦克风供电 |
| GND | GND | 共地 |
| SCK / BCLK | GPIO5 | I2S 麦克风时钟 |
| WS / LRCK | GPIO4 | I2S 左右声道时钟 |
| SD / DOUT | GPIO6 | 麦克风数据输入 |
| L/R | GND | 固定声道选择 |

当前固件中的对应定义：

```c
#define AUDIO_I2S_MIC_GPIO_WS   GPIO_NUM_4
#define AUDIO_I2S_MIC_GPIO_SCK  GPIO_NUM_5
#define AUDIO_I2S_MIC_GPIO_DIN  GPIO_NUM_6
```

## 功放接线

MAX98357A 使用 5 V 供电，I2S 音频信号仍由 ESP32-S3 的 3.3 V GPIO 输出。

| MAX98357A | ESP32-S3 | 说明 |
| --- | --- | --- |
| VIN / VCC | 5V / 5Vin | 功放供电 |
| GND | GND | 共地 |
| BCLK | GPIO15 | I2S 功放时钟 |
| LRC / LRCK | GPIO16 | I2S 左右声道时钟 |
| DIN | GPIO7 | 功放音频数据输入 |
| SD / EN | 3V3 | 拉高使能功放 |
| GAIN | 悬空 | 使用模块默认增益 |
| SPK+ / SPK- | 喇叭正负极 | 只接喇叭，不接 ESP32-S3 GND |

当前固件中的对应定义：

```c
#define AUDIO_I2S_SPK_GPIO_DOUT GPIO_NUM_7
#define AUDIO_I2S_SPK_GPIO_BCLK GPIO_NUM_15
#define AUDIO_I2S_SPK_GPIO_LRCK GPIO_NUM_16
```

## 共地说明

共地就是把所有模块的 `GND` 连到同一个地参考上：

- ESP32-S3 的 GND
- INMP441 的 GND
- MAX98357A 的 GND
- 如果功放使用外部 5 V 电源，也要把外部电源的 GND 接到 ESP32-S3 的 GND

I2S 的时钟和数据线都以 GND 作为电平参考。没有共地时，常见现象是麦克风无输入、功放无声、噪声很大，或者设备表现不稳定。

注意：MAX98357A 的 `SPK-` 是功放喇叭输出端，不等于系统 GND，不要把 `SPK-` 接到 ESP32-S3 的 GND。
